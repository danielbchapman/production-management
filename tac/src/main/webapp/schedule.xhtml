Schedule Module

City Management

The CITY MANGMENT provides an interface for managing information about cities including the hotels, hospitals, points of interest and other basic information. This is related to the CALENDAR as it will be updated with the information in this interface.

Calendar

The CALENDAR provides an interface for managing the entire tour schedule including the venue, hotel and call time for various events. It also allows the generation of a PDF so that it can be easily distributed.

Performances

PERFORMANCES provides an interface for managing the length and properties of a performance. This allows the calendar to be rapidly updated to adjust for increased call times or other various time factors. Generally speaking it is used to establish an estimated strike in and out time.
